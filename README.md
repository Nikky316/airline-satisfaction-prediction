# ✈️ Airline Customer Satisfaction Prediction

## 📊 Overview
This project uses Logistic Regression to predict airline passenger satisfaction.

## 🔧 Steps
- Data preprocessing
- Handling missing values
- Feature encoding
- Model training (Logistic Regression)

## 📈 Evaluation
- Confusion Matrix
- Precision
- Recall

## 🔍 Key Insights
- In-flight entertainment improves satisfaction
- Flight delays reduce satisfaction
- Wi-Fi quality impacts customer experience

## 💡 Recommendations
- Improve Wi-Fi service
- Reduce delays
- Enhance onboard experience

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
